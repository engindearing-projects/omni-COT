# OmniCOT v0.1-alpha Release Notes

Release Date: November 4, 2025
Status: Pre-release (Alpha)
Build: TAK Third-Party Pipeline

## Release Information

First alpha release built through TAK third-party pipeline.

APK: ATAK-Plugin-omnicot-0.1--5.4.0-civ-release.apk (signed)

## Features

### CoT Management
- Modify marker affiliations and battle dimensions
- Real-time CoT event federation
- Standard ATAK CoT dispatcher integration

### AOI Detection
- Automatic detection of areas of interest from map shapes
- Area calculation and display
- Center coordinate extraction

### Dashboard Interface
- Modern card-based UI
- Real-time metrics display
- Activity tracking

### Alert System Foundation
- Infrastructure for location-based alerts
- Geofencing preparation

## Installation
1. Download ATAK-Plugin-omnicot-0.1--5.4.0-civ-release.apk
2. Install on device with ATAK 5.4.0
3. Enable plugin in ATAK Settings > Plugin Management
4. Restart ATAK
